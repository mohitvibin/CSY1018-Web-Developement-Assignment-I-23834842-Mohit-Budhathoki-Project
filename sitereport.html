Introduction


This HTML code defines a webpage with a navigation bar containing links to different pages. It uses the Font Awesome library for icons and a separate style.CSS file for styling. The webpage also includes a section with a brief introduction and a call-to-action button. Problems and Challenges


I designed this portfolio and tried my best to design with possible information and features as much as possible. There are few potential problems and challenges that I faced while making this portfolio.
Some of them are:
Linking external resources: The code is linking to two external resources, the Font Awesome stylesheet and the custom stylesheet. If the links are broken or the resources become unavailable, it could cause issues with the layout and styling of the website.
Code organization: The code is not well-organized, which could make it difficult to maintain and update in the future.
